CMAS open-code

A free online platform for sharing movement analysis code and resources

Welcome to the CMAS open-code project!

Our aim is to share coding resources, to improve and accelerate movement analysis.

We would like our platform to work for everyone:

  • If you already have technical skills, you can collaborate, create new tools, and share them with others.
  • If you are a coding beginner, you will find a library of tutorials and example code to support your learning.
  • If you would like to use existing code, you will be able to easily find relevant resources, with instructions and examples of use.
